# Generate Legal Examples Workflow

## Doel

Deze workflow creëert begrijpelijke praktijkvoorbeelden van juridische concepten uit de LegalOntology. Het maakt abstracte ontologische concepten toegankelijk door middel van concrete, realistische scenario's die geschikt zijn voor verschillende doelgroepen.

## Gebruik

Activeer deze workflow door in de LegalAnalyst agent te typen: `*generate-examples`

## Invoer

De workflow vraagt om specificaties voor voorbeeldgeneratie:
1. **Juridische concepten** - Namen of selectie uit LegalOntology
2. **Doelgroep** - Juristen, burgers, experts, studenten
3. **Complexiteitsniveau** - Basis, gemiddeld, geavanceerd
4. **Aantal voorbeelden** - 1-5 per concept
5. **Context domein** - Privacyrecht, contractenrecht, administratief recht, etc.

## Uitvoer

Genereert een georganiseerde collectie praktijkvoorbeelden met:
- Individuele concept illustraties
- Progressieve complexiteitsopbouw
- Stakeholder perspectieven
- Educatieve uitleg en implicaties
- Gebruiksaanwijzingen voor verschillende contexten

## Voorbeeld Typen

### Scenario Categorieën
- **Hypothetische scenario's** - Geïdealiseerde situaties
- **Reële wereld aanpassingen** - Geanonimiseerde praktijkgevallen
- **Progressieve complexiteit** - Opbouwende moeilijkheidsgraden
- **Cross-domein toepassingen** - Concepten in verschillende contexten

### Audience Aanpassingen
- **Burgers**: Simpele taal, dagelijkse situaties, praktische consequenties
- **Studenten**: Educatieve uitleg, leerpunten, studievoorbeelden
- **Juristen**: Professionele terminologie, juridische gevolgen, praktijkcases
- **Experts**: Diepgaande analyse, edge cases, academische relevantie

## Bestanden

- `workflow.yaml` - Configuratie met audience parameters
- `instructions.md` - 8-staps voorbeeld generatie methodologie
- `template.md` - Uitgebreide voorbeeld collectie structuur
- `checklist.md` - Educatieve en ethische validatie criteria

## Pedagogische Methodologie

Deze workflow gebruikt:
- **Audience-centered design** - Voorbeelden aangepast op kennisniveau
- **Progressive disclosure** - Informatie geleidelijk onthullen
- **Multi-perspective analysis** - Verschillende stakeholder gezichtspunten
- **Cross-contextual learning** - Concepten in diverse scenario's

## Ethiek en Accurateit

De workflow waarborgt:
- **Ethiek**: Correcte anonimisering, geen misleidende informatie
- **Accurateit**: Conceptuele trouw aan LegalOntology
- **Diversiteit**: Culturele en contextuele variatie
- **Validatie**: Professionele review vereist voor praktijktoepassing

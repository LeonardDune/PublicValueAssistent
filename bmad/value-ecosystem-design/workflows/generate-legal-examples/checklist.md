# Validatie Checklist: Juridische Voorbeelden Generatie

## Structuur en Compleetheid

- [ ] Parameters correct ingesteld (audience, complexity, aantal)
- [ ] LegalOntology concepten correct geselecteerd en geladen
- [ ] Voorbeeld kaders gedefinieerd (scenario types, complexiteit levels)
- [ ] Alle concepten hebben minimaal opgegeven aantal voorbeelden

## Ontologische Accuraatheid

- [ ] Voorbeelden correleren correct met LegalOntology concepten
- [ ] UFO stereotypen correct weergegeven in praktijkscenario's
- [ ] Ontologische relaties accuraat geïllustreerd
- [ ] Geen contradicties met juridische principes

## Educatieve Kwaliteit

- [ ] Voorbeelden begrijpelijk voor target audience
- [ ] Juiste complexiteitsniveau gehanteerd
- [ ] Duidelijke uitleg van juridische concepten
- [ ] Praktische toepasbaarheid aangetoond

## Voorbeeld Diversiteit en Realisme

- [ ] Verschillende voorbeeld types gebruikt (hypothetisch, real-world)
- [ ] Stakeholder perspectieven opgenomen
- [ ] Culturele en contextuele diversiteit
- [ ] Realistische scenario's (geen onmogelijke situaties)

## Ethiek en Accuraatheid

- [ ] Voorbeelden ethisch verantwoord (geen misleidende informatie)
- [ ] Gevoelige informatie correct geanonimiseerd
- [ ] Juridische accuraatheid gegarandeerd
- [ ] Geen valse verwachtingen gewekt

## Technische Kwaliteit

- [ ] Voorbeelden consistent geformatteerd
- [ ] Template variabelen volledig vervangen
- [ ] Voorbeelden traceerbaar naar concepten
- [ ] Index en categorisatie compleet

## Knowledge Graph Integratie

- [ ] CRITICAL: Bestaande legal-analysis-graph.ttl correct GELADEN
- [ ] Voorbeelden toegevoegd als gUFO compliant triples (:LegalExample, gufo:typeOf gufo:Situation)
- [ ] Links naar LegalOntology concepten correct ingesteld
- [ ] SAVE uitgevoerd naar knowledge/legal-analysis-graph.ttl
- [ ] Educatieve metadata triples opgenomen

## gUFO Compliance Validatie

- [ ] Alle triples conforme gUFO specificatie (rdf:type + gufo:typeOf)
- [ ] Verwijzingen naar OntologyBasis.ttl concepten
- [ ] Voorbeeld triples hervindbaar voor future queries
- [ ] Semantic links tussen voorbeelden en concepten

## Validatie en Naleving

- [ ] Cross-referenties naar gerelateerde concepten correct
- [ ] Voorbeelden onafhankelijk valideerbaar
- [ ] Publiek domein scenario's gebruikt waar mogelijk
- [ ] Disclaimer voor educatieve doeleinden opgenomen

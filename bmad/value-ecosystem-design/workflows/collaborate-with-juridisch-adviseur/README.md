# Collaborate with Juridisch Adviseur Workflow

## Doel

Deze workflow faciliteert gestructureerde samenwerking tussen ontologie experts en juridisch adviseurs voor professionele advisering op public service propositions.

## Gebruik

Activeer deze workflow door in de LegalAnalyst agent te typen: `*integrate-advisor`

## Invoer

De workflow vraagt om samenwerkingspecificaties:
1. **Type advies** - Juridisch, ethisch, technisch, geïntegreerd
2. **Public service proposition** - Beschrijving van het voorstel
3. **Urgentieniveau** - Hoog, medium, laag voor prioritering
4. **Expertise gebieden** - Specifieke juridische domeinen
5. **Communicatievoorkeuren** - Online, fysiek, schriftelijk

## Uitvoer

Genereert een uitgebreide samenwerkingsdocumentatie met:
- Contextuele adviesvoorbereiding voor adviseur
- Gestructureerde adviesverzoeken
- Consultatie proces verslaglegging
- Expert analyse integratie
- Uitvoerbare aanbevelingen voor implementatie

## Expert Integratie Proces

### Voorbereiding Fase
- Context verzameling voor juridische expertise
- Advies verzoek formulering met specifieke vragen
- Stakeholder belangen mapping

### Consultatie Fase
- Adviseur scheduling en communicatie facilatie
- Proces documentatie tijdens consultatie
- Real-time analyse capturing

### Integratie Fase
- Expert input synthese met project context
- Risico identificatie en mitigatie strategieën
- Compliance aanbevelingen ontwikkeling

## Professionele Workflow

### Juridische Beoordeling
- Haalbaarheidsanalyse van proposities
- Compliance vereisten identificatie
- Risico assessment en mitigatie

### Implementation Guidance
- Juridisch compliant oplossingsontwikkeling
- Monitoring en audit procedures
- Documentatievereisten specificatie

## Toepassingsgebieden

### Public Service Ontwikkeling
- Juridische compliance van nieuwe diensten
- Wettelijke haalbaarheidsstudies
- Risico assessment voor implementatie

### Beleidsondersteuning
- Juridische implicaties van beleid
- Stakeholder impact analyses
- Compliance route mapping

### Project Governance
- Juridisch advies integratie in decision making
- Compliance monitoring systemen
- Audit en rapportage kaders

## Professionele Borging

De workflow verzekert:
- **Structured communication** tussen ontologie experts en juristen
- **Professional documentation** van alle advies interacties
- **Compliance integration** in ontwerp beslissingen
- **Risk mitigation guidance** voor implementation

## Vereisten

- Toegang tot LegalOntology voor context
- Juridisch adviseur beschikbaarheid (intern/extern)
- Public service proposition beschrijving
- Stakeholder mapping compleet

## Uitvoeringsrichtlijnen

Na workflow finalisatie:
1. Distribueer adviesrapport aan relevante stakeholders
2. Implementeer aanvaarde recommendaties
3. Stel follow-up consultaties in volgens roadmap
4. Monitor compliance tijdens implementatieperiode
